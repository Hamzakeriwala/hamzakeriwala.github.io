by The SQL Injectors - Junaid, Nadeem, Minsoo, Andres and Alex This project was part of the Introduction to Software engineering (CSCC01) course. We were put into a team of 5 people and had to develop a plugin for an existing software called Zotero. This plugin was based on requirements provided by a client, a professor from our university (UofT). As part of the project, we followed the AGILE software development method by creating burn down charts based on user stories and having regular meetings, etc. Eventually, we ended up developing a fine plugin that takes care of batch-editing of tags and being able to modify custom output styles. The following is our read me page for this project, from Github.  

Installation

Simply install ZoteroEXT.xpi addon file to Zotero standalone or firefox.

You may also choose to build it yourself:

Manually:
  1. Clone repo
  2. zip the ‘chrome’ folder, chrome.manifest, and ‘install.rdf’
  3. change the .zip extension to ‘.xpi’
Or using our script:
Windows: (from a command prompt) generate_xpi.bat Linux/Mac: sh generate_xpi.sh

Features

Intro video (Please Click to watch!):
  • Batch add, remove, rename, or merge multiple tags in one go!
  • Download, edit, and install user made CSL styles with ease using our built in powerful csl-editor!
